1. Сборка через Apache Maven
   mvn package

2. Сборка располагается ./money-service-loader/target/
   conf/application.properties  		    пример конфигурационного файла
   conf/log4j.properties			        пример файла для логгера
   money-service-loader-0.0.1.jar		    собранный артифакт

3. Запуск модуля осуществляется скриптом runService.bat

4. Пример запроса : https://<host>:<port>/send?xml=<xmlText>
